# Cronus Windows & Doors Website

This repository contains the source code for the Cronus Windows & Doors marketing website. The design focuses on conveying professionalism, trust and value for homeowners considering window and door replacement. The site is lightweight, fully responsive and ready to deploy on GitHub Pages.

## Structure

- **index.html** – Primary landing page featuring hero section, reasons to choose Cronus, services overview, financing info, about, gallery, reviews and contact.
- **windows.html** – Dedicated service page detailing our window replacement offerings, benefits and call to action.
- **doors.html** – Dedicated service page detailing our entry and patio door offerings, benefits and call to action.
- **style.css** – Central stylesheet defining the colour palette, typography, layout and responsive styling.
- **script.js** – Small enhancements, such as updating the copyright year dynamically.
- **assets/images/** – Contains the logo and placeholder imagery used across the site.

## Deployment

1. **Upload** the contents of this repository to your GitHub repository named `cronus-windows-doors`.
2. In the repository settings, enable **GitHub Pages** to serve from the `main` branch and the root directory.
3. After a few minutes, your site will be available at `https://your-username.github.io/cronus-windows-doors/`.

## Customization

- Replace the placeholder images in `assets/images/` with photos of your actual projects for authenticity.
- Update the contact information, service area and review links as needed.
- Add or edit content in the dedicated service pages for additional products (e.g. storm doors, impact windows, etc.).
- Integrate Google Reviews or Facebook Reviews by replacing the placeholder review section with real data.
